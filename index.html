O que eu aprendi na aula de HTML

Na minha primeira aula de HTML, fiz meu primeiro projeto de blog com as tags aprendidas no curso da DIO.me

Veja a seguir como foi cada aula do curso inicial de HTML da DIO

  • Módulo 1 - Comunicação Client-Server
  • Módulo 2 - Introdução prática ao HTML
  • Módulo 3 - Entrega de projeto

Módulo 1 - Comunicação Client-Server

(voltar)

As primeiras aulas, como todo curso complexo, são feitas para mostrarem a teoria por trás de algo que você fará: da história das primeiras máquinas de criptografia até os computadores atuais, o que é o protocolo TCP/IP, o que são os clientes (aqueles que usam a internet para adquirir a informação dos servidores), o que são os servidores (os lugares na internet que armazenam tais informações), aplicações web, hospedagem e, claro, aprendemos que HTML não é uma linguagem de programação, apenas uma linguagem de marcação.

Módulo 2 - Introdução prática ao HTML

(voltar)

As aulas seguintes serviram pra mostrar como funciona o HTML na prática. Do download do Visual Studio Code às primeiras tags do HTML, fomos guiados pelos nossos primeiros passos para montar algo prático na internet. Começamos com tags para modificar a escrita, como a tag i, que transforma palavras em itálico, a tag strong, que transforma palavras em negrito e a tag u, que sublinha as palavras.

Também aprendemos acerca dos atributos das tags, como fazer listas ordenadas - com numeração - e não ordenadas - sem numeração, além de inserir links internos e externos nas nossas páginas.

Depois de muitas horas de aprendizado teórico e prático, foi a vez de entregar meu primeiro projeto prático, este que você está lendo agora.
